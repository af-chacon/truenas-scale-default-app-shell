Icon Files for TrueNAS Shell Selector Extension

This directory contains the icon files used by the extension:

- icon-19.png: Browser action icon (19x19 pixels)
- icon-38.png: Browser action icon for retina displays (38x38 pixels)
- icon-48.png: Extension icon for the Add-ons Manager (48x48 pixels)
- icon-96.png: Extension icon for the Add-ons Manager on retina displays (96x96 pixels)

If you see placeholder icons or warnings about corrupted icons, you can:
1. Run `npm run fix-icons` to create basic placeholders
2. Replace these with custom icons by manually creating proper PNG files
   at these sizes

Icon design guidelines:
- Use simple, recognizable imagery
- Maintain good contrast for visibility
- Design should scale well across different sizes
- Use transparency where appropriate
- Follow Firefox Add-on icon guidelines
